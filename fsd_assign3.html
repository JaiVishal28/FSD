My Projects

OnlyLoans – Loan Management System (Python, Tkinter, MySQL)

OnlyLoans is a desktop-based Loan Management System developed using Python with Tkinter for the GUI and MySQL for the backend. The application manages end-to-end loan processing and customer information through a clean, user-friendly interface. It features five interconnected tables for users, loans, payments, interest tracking, and loan status. Key functions include adding and updating customer records, calculating EMIs, generating reports, and maintaining repayment history. This project strengthened my understanding of Database Management Systems (DBMS), SQL query optimization, and GUI development with Python.

VTU PDF Reader – Android App (Android Studio, Java)

This project is a dedicated Android app designed to help VTU (Visvesvaraya Technological University) students quickly access their semester PDFs. Built using Android Studio and Java, the app provides a smooth and intuitive interface for browsing, downloading, and viewing syllabus, question papers, and academic resources—all within the app. I implemented PDF rendering using third-party libraries, file caching, and basic offline capabilities. This project helped me improve my mobile UI/UX design, Android lifecycle management, and Java development skills.
